Add age verification popup for WooCommerce products with customizable minimum and maximum age requirements.

== Description ==

ThaiTop Age Verification Popup adds a customizable age verification popup for WooCommerce products. Perfect for stores selling age-restricted products.

Features:
* Individual product age verification
* Customizable minimum and maximum age requirements
* Current or custom date age calculation
* Buddhist year display support (optional)
* Multiple color templates (Steam, Dark, Light, Nature)
* Full color customization
* Mobile-responsive design
* Multi-language support
* WooCommerce HPOS compatible

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/thaitop-age-verification-popup`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WooCommerce > Settings > Age Verification to configure display options
4. Edit any product to enable and configure age verification requirements

== Frequently Asked Questions ==

= Can I set different age requirements for different products? =
Yes, age verification can be configured individually for each product.

= Does it support other calendars? =
Currently supports Gregorian calendar with optional Buddhist year display.

= Is it compatible with WooCommerce HPOS? =
Yes, the plugin is fully compatible with WooCommerce High-Performance Order Storage.
