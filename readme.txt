Add age verification popup for WooCommerce products with customizable minimum age requirements.

== Description ==

ThaiTop Age Verification Popup adds a customizable age verification system to your WooCommerce products. Perfect for stores selling age-restricted products.

Key Features:

* Individual product age verification
* Customizable minimum age requirements
* Current or custom date age calculation
* Buddhist year display support (optional)
* Multiple color templates (Steam, Dark, Light, Nature)
* Full color customization
* Mobile-responsive design
* Multi-language support
* WooCommerce HPOS compatible

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/thaitop-age-verification-popup` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure the plugin settings in WooCommerce > Settings > Age Verification

== Frequently Asked Questions ==

= Is WooCommerce required? =

Yes, this plugin requires WooCommerce 6.0 or higher to be installed and activated.

= Can I customize the popup design? =

Yes, you can choose from predefined templates or customize colors to match your theme.
